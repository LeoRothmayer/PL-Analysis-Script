# PL-Analysis-Script

Python GUI for analyzing/normalizing Photoluminescence data with multiple measurement windows.

## Important/Disclaimer

The mathematical/functional basis of this script was written by myself, the GUI and further features were written using Claude Code. The initial Version was generated by Thomas Trinkl, further improvements and the current version by me.

## General Functionality

 

### I. Loading files

  
- Files can be loaded as individual files or entire folders (preferably as .origin file)
- load halogen reference, dark measurements (background), whitelight measurement and actual PL spectra

### II. Power-by-Power Pipeline

Main focus, manual editing choices for each spectrum and individual stitching

### III. Automatic Mode

Automatic analysis with minimal manual input - all spectra will be edited at once - faster but less precise
